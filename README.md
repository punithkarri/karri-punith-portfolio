# Karri Punith - AI/ML Engineer & Full Stack Developer Portfolio

Welcome to the source code for my personal portfolio website! This project showcases my skills, experience, and projects in AI/ML engineering and Full Stack development.

## 🌐 Live Website
Check out the deployed version of my portfolio here: **[https://karri-punith.vercel.app/](https://karri-punith.vercel.app/)**

## 🚀 About the Project
This portfolio is built with HTML, CSS, and JavaScript, focusing on a clean, modern, and dark-themed UI. It features:
- **Bento Grid Layout**: A modern approach to structuring content, especially in the Hero and About sections.
- **Interactive Elements**: Custom cursor glow, smooth scrolling, and scroll-triggered animations.
- **Custom AI Chatbot Assistant**: An interactive chatbot integrated into the UI.
- **Responsive Design**: fully functional on desktop and mobile devices.

## 🛠️ Tech Stack
- **HTML5**: Semantic structure
- **CSS3**: Custom variables, Grid/Flexbox layouts, Animations
- **Vanilla JavaScript**: DOM manipulation, Chatbot logic, UI interactions
- **FontAwesome**: Icons
- **Google Fonts**: Inter & Space Grotesk

## 📂 Project Structure
- `index.html` - The main structure of the webpage.
- `style.css` - All styling including layout, colors, typography, and animations.
- `script.js` - Interactions, scroll animations, and chatbot functionality.
- `assets/` - Images, certificates, and media assets used in the site.

## 💡 Key Sections
- **Hero / Bento Grid**: Introduction, roles, and quick links.
- **About / Tech Stack**: Detailed background and skills showcase.
- **Work (Portfolio)**: Real-world projects like *Freshin10*, *Library Tracker*, and *Crop Disease Detection*.
- **Experience & Education**: Timeline of internships, training, and education.
- **Certifications**: Masonry grid of various certificates earned.
- **Contact**: Let's connect! Email, LinkedIn, GitHub, and a contact form.
